Task 10 - Creating and Using SQL Views

This task demonstrates how SQL views simplify complex queries and improve security.

Concepts Covered:
- Creating views from JOIN queries
- Querying views like tables
- Filtering and sorting view data
- View-based access control
- Limitations of updating views
- Dropping and recreating views
- Views for reporting and dashboards

Outcome:
Able to design reusable and secure query layers using SQL views.
